UMP Merch Django project

e-commerce platform

An e-commerce platform developed for the University of Mpumalanga, offering a range of supporter kits, memorabilia, and exclusive merchandise for students, staff, and alumni. The website provides a seamless shopping experience with user-friendly navigation and responsive design. It integrates a secure payment gateway (PayFast) to facilitate easy transactions, and the platform is optimized for mobile devices and future scalability. Built using Django, the site leverages SQLite for database management, with plans to migrate to a cloud-friendly database. The admin side is enhanced with Jazzmine for a customized and efficient backend interface.
